# Cyber World

以瀏覽器遊玩的賽博都市角色扮演遊戲。包含五人即時戰鬥、隨機打寶、裝備強化、商店、夜生活，以及可購買並經營產業的被動收入系統。

## 遊玩

開啟 GitHub Pages 網址即可遊玩。遊戲進度與玩家上傳的角色肖像會保存在目前裝置的瀏覽器中，不會上傳到伺服器。

## 操作提示

- 在「人物」頁可修改隊員名稱與肖像。
- 在「戰鬥」頁選擇地區打寶，裝備會有隨機品質、屬性與數值。
- 在「裝備」頁使用武卷、防卷與保護卷強化裝備。
- 在「城市」與「產業」頁安排生活活動、購買資產、聘請經理並領取被動收入。

本作仍會持續擴充，建議使用最新版 Chrome、Edge 或 Safari。
